FAVOR-Bench

A Comprehensive Benchmark for Evaluating Large Language Models

Paper GitHub Hugging Face

Abstract

FAVOR-Bench is a comprehensive benchmark designed to evaluate large language models across multiple dimensions including factuality, reasoning, understanding, and robustness. Our benchmark provides a standardized way to assess model performance and helps identify strengths and weaknesses in current state-of-the-art language models.

Methodology

FAVOR-Bench evaluates models across four key dimensions:

  • Factuality: Assessing if model outputs contain accurate information
  • Adaptability: Measuring model performance across diverse domains
  • Versatility: Evaluating models on different types of reasoning tasks
  • Robustness: Testing model stability against adversarial inputs
